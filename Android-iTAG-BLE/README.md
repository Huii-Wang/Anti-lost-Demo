# 安卓 iTAG BLE DEMO

## 1.代码思路
    1.检查及初始化相关设置 api版本大于等于19 需要使用权限 蓝牙权限 定位权限
    2.发起扫描，对扫描结果进行筛选
    3.连接目标设备
    4.监听通知服务，获取到BLE发送到手机上的消息（接受防丢器按键消息）
    5.发送消息，给防丢器发送报警/取消报警命令
    6.断开APP防丢器BLE的连接



## 2.Demo使用方法
    1.申请定位权限
    2.防丢器开机，然后进行扫描设备，设备扫描成功会提示：发现目标设备
    3.点击链接设备
    4.发送报警命令 开始响铃
    6.发送停止报警命令，关闭响铃
